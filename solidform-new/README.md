# Solid Form Pty Ltd - Construction Company Website

Professional website for Solid Form Pty Ltd - Formwork Contractors & Labour Hire.

## 🚀 Deploy to Netlify (100% Working!)

### Method 1: Drag & Drop (Easiest - 60 seconds!)

1. Build the site:
   ```bash
   npm run build
   ```

2. Go to [netlify.com/drop](https://app.netlify.com/drop)

3. Drag the `build` folder to the upload area

4. Done! You'll get an instant live URL

### Method 2: GitHub Integration

1. Push this code to GitHub

2. Go to [Netlify](https://app.netlify.com)

3. Click "Add new site" → "Import an existing project"

4. Connect your GitHub repository

5. Settings will auto-detect:
   - Build command: `npm run build`
   - Publish directory: `build`

6. Click "Deploy"
